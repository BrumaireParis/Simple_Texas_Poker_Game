## 德州扑克游戏

这是一个 Python + Tkinter 编写的简易德州扑克图形界面版本。

## 功能

- 主菜单：新游戏、排行榜、设置、退出游戏。
- 设置：可选择主流窗口分辨率、UI 缩放比例，也可切换全屏。
- 多语言：支持中文、英语、德语、法语、西语、葡语、日语、韩语；首次启动默认跟随操作系统语言，可在设置中切换。
- 新游戏：可选择 AI 数量、AI 难度和分房，最多 6 人同桌（玩家 + 5 名 AI）。
- 分房：新手房、普通房、高倍房，盲注和初始资金不同。
- 玩法：庄位轮转、小盲/大盲、烧牌、翻牌/转牌/河牌、看牌、跟注、加注、弃牌、摊牌、all-in 分池结算。
- AI：不同难度使用不同策略，高级 AI 会通过蒙特卡洛模拟估算胜率；每名 AI 会随机获得紧凶、松凶、跟注型等隐藏游玩风格。
- AI 名字：会按当前语言随机使用常见姓名，并使用当前语言的 AI/KI/IA 等后缀；游戏术语、日志、牌型、房间和难度会随语言本地化。
- 对局节奏：AI 会随机思考一段时间再行动；玩家行动有 30 秒倒计时。
- 牌桌表现：AI 使用随机姓名和虚拟头像，座位名显示为“姓名 (AI)”；AI 行动会以浮层提示短暂显示并淡出。
- 界面信息：侧边栏重点信息固定显示，行动记录支持滚轮滚动；底池和玩家资金会用筹码堆图像表现。
- 复盘辅助：摊牌后，赢家最佳 5 张牌会用黄色边框高亮。
- 结算：AI 资金输完会离开牌桌；玩家资金输完游戏结束。
- 排行榜：按净收益优先排序，显示资金、胜局/手数、胜率和房间；空榜会显示虚拟挑战记录，首次保存真实成绩后自动替换。

## 运行方式

确保本机已安装 Python 3，然后在当前目录运行：

```bash
python texas_holdem_gui.py
```
